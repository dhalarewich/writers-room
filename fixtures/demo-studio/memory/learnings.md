# Learnings

Rules distilled from real corrections. Newest first, above the marker. Superseded rules get struck through with a pointer, never deleted. `/learn --fold` graduates stable rules into [[style-dna]] or [[banned-patterns]] quarterly.

Entry format:

### YYYY-MM-DD — one-line rule
- **Context**: card id — what happened
- **Evidence**: the operator words or edit-diff that produced this
- **Apply to**: writer | editor | warden | critic | muse | scout
- **Status**: active

### 2026-06-20 — Never convert a field number into a rounded marketing number
- **Context**: fn-0006 (earlier draft) — "nearly 3×" was hand-corrected back to "2.7×"
- **Evidence**: edit diff: `- nearly triples repeat purchases` → `+ repeat-purchase rate we measured at 2.7×`
- **Apply to**: writer, editor
- **Status**: active

### 2026-06-02 — Cost findings land as "what it costs you at 2am," not as percentages
- **Context**: fn-0004 sketch feedback — operator reframed the fuel math from % overhead to grams-per-day carried
- **Evidence**: operator note: "nobody feels a percent. they feel 40 extra grams every day"
- **Apply to**: writer, muse
- **Status**: active
